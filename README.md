# RSS aggregator

A simple website that allows you to load and read RSS feed. RSS aggregator is hosted on [Vercel](https://frontend-project-11-silk-eight.vercel.app/)

## Setup

```sh
make install
```

## Build

```sh
make build
```

## Run

```sh
make dev
```
